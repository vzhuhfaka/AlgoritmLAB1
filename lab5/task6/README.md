## Задание

Реализуйте очередь с приоритетами. Ваша очередь должна поддерживать следующие операции: добавить элемент, извлечь
минимальный элемент, уменьшить
элемент, добавленный во время одной из операций.

## Input / Output

| Input | Output |
|-------|--------|
| 8     | 2      | 
| A 3   | 1      |
| A 4   | 3      |
| A 2   | *      |
| X     |        |
| D 2 1 |        |
| X     |        |
| X     |        |
| X     |        |
